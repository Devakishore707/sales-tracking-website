Supabase Database Initial Setup Guide

To enable sync, you need to create the table in your Supabase project. Copy the SQL code below, go to your Supabase SQL Editor, paste it, and click Run:

-- 1. Create the sales table create table if not exists sales ( no bigserial primary key, product_name text not null, category text not null, price numeric(12, 2) not null, quantity integer not null, total numeric(12, 2) not null, created_at timestamp with time zone default timezone('utc'::text, now()) not null ); -- 2. Enable Row Level Security (RLS) alter table sales enable row level security; -- 3. Create policies for anonymous access create policy "Allow public inserts" on sales for insert to anon with check (true); create policy "Allow public selects" on sales for select to anon using (true); create policy "Allow public deletes" on sales for delete to anon using (true);
